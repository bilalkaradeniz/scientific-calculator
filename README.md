# Bilimsel Hesap Makinesi

HTML, CSS ve JavaScript kullanılarak geliştirilmiş modern, responsive ve bilimsel hesap makinesi projesi.

## Özellikler

- Normal mod
- Bilimsel mod
- Dark / Light tema desteği
- Tema kaydetme (localStorage)
- Hafıza sistemi (M+, MR, MC)
- İşlem geçmişi
- Klavye desteği
- Mobil uyumlu tasarım
- Modern animasyonlu arayüz
- Cam efekti (Glassmorphism tasarım)

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
